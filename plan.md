# 《天麻的旅程》原料溯源交互动画 / 一页式网页策划稿

## 一、页面总定位

### 页面名称

**天麻的旅程：从昭通小草坝到月中桂烘焙车间**

### 一句话体验

用户向下滑动页面，跟随 IP 角色“小安”穿过昭通小草坝的云雾深山、天麻初加工工坊、月中桂百年糕点车间，最终抵达手中的天麻酥礼盒。

### 核心传播目的

这不是一个单纯介绍产品的网页，而是一个“建立信任”的溯源体验页。

消费者对天麻类食养产品的核心顾虑通常有三点：

1. **原料是不是真的好？**
2. **天麻是不是安全、合规、能作为食品吃？**
3. **老字号做新产品，是噱头还是有真实技艺支撑？**

所以页面要用“视觉化旅程”替代大段说教，让用户亲自完成一次从产地到产品的路径验证。

### 主叙事线

**一味昭通天麻，穿过云雾山林，经过精细炮制，走进月中桂百年糕点技艺，最终成为一口可以带回家的东方食养点心。**

### 情绪曲线

1. **第一章：自然感、神秘感、原生态信任**
2. **第二章：洁净感、工艺感、安全感**
3. **第三章：温度感、匠心感、老字号文化感**
4. **第四章：产品感、购买冲动、礼赠价值感**

---

# 二、整体页面结构

1. **首屏引入：跟随小安，开启一味天麻的旅程**

   * 屏 10：礼盒展开

# 三、关键视觉设定

## 1. 画风建议

### 整体风格

**东方手绘插画 + 轻 3D 景深 + 现代食品品牌质感。**

不要做成纯科普页面，也不要做得过于卡通。最理想的感觉是：有绘本的温度，也有食品品牌的高级感。

### 四章视觉调性

#### 第一章：深山采挖

* 色调：青绿、雾白、土褐。
* 氛围：云雾、湿润、森林、苔藓、山风。
* 关键词：原生、自然、冷凉山地、林下生长。

#### 第二章：古法炮制 / 精细加工

* 色调：米白、浅木色、淡金色、洁净灰。
* 氛围：干净、有序、工艺流程清楚。
* 关键词：清洗、分拣、切片、烘干、粉碎、细度。

#### 第三章：百年手艺

* 色调：暖黄、麦色、木色、烘焙棕。
* 氛围：灯光温暖、手作、有烟火气。
* 关键词：老字号、老师傅、揉面、包酥、焙烤。

#### 第四章：到你手中

* 色调：品牌主色 + 礼盒色 + 金色点缀。
* 氛围：高级、礼赠、完成感。
* 关键词：礼盒、信任标签、尝鲜按钮、溯源码。

# 首屏：开启旅程

## 屏幕目标

让用户立刻理解：这是一个可滚动的溯源旅程，不是普通产品详情页。

## 画面描述

全屏云雾从山间缓慢流动。远处是昭通小草坝的山林轮廓，近处有一条蜿蜒小路。小安站在画面下方偏右，背着小竹篓，回头看向用户。

画面中央出现标题：“天麻的旅程”。副标题逐字浮现：“从昭通小草坝，到月中桂的一口酥香。”

底部有一个向下滚动提示：“向下滑动，跟随小安出发”。

## 主文案

**天麻的旅程**

## 副文案

**从昭通小草坝的云雾深山，到月中桂的百年烘焙车间。**

## 小安对白

“准备好了吗？我们去看看，一块天麻酥从哪里来。”

## 交互设计

* 鼠标轻微移动时，前景树叶产生视差移动。
* 向下滚动后，小安开始沿小路前进。
* 点击“小安”可以出现一句轻提示：“每一步，都有迹可循。”

## 动效分镜

1. 0—1 秒：云雾淡入，山体轮廓出现。
2. 1—2 秒：标题从雾中浮现。
3. 2—3 秒：小安从画面侧边走入。
4. 用户滚动：镜头缓慢推进山林，小安保持在视觉中心附近。

## 所需素材

* 昭通小草坝山林手绘主背景。
* 云雾 SVG / PNG 分层素材。
* 小安角色正面、侧身、回头三种姿态。
* 向下滚动箭头动效。
* 背景环境音可选：山风、鸟鸣、轻微铃声。

# 第一章：深山采挖

## 章节定位

建立原料信任：让用户相信这不是普通天麻，而是来自特定产区、特定生态环境中的优质天麻。

## 章节主标题

**第一章 · 深山采挖**

## 章节主文案

**好原料，先要经得起山林的检验。**

---

## 屏 1：云雾小草坝

### 画面描述

镜头进入山林深处。画面上方是云雾缭绕的山坡，中景是高大树木，前景有苔藓、落叶、湿润土壤。小安沿山路走来，在一块写着“小草坝”的木牌前停下。

木牌上可以出现：“昭通 · 彝良 · 小草坝”。

### 主文案

**在昭通小草坝，天麻先学会与山林共生。**

### 辅助文案

**冷凉山地、云雾滋养、林下环境，让这里成为昭通天麻的重要产区之一。**

### 信息点文案

信息点 1：**冷凉山地**
文案：山地气候带来昼夜温差，也让天麻的生长更慢、更扎实。

信息点 2：**云雾森林**
文案：湿润环境、林下土壤与菌材，共同构成天麻适宜的生长条件。

信息点 3：**昭通天麻**
文案：昭通天麻是云南具有代表性的地理标志产品之一。

### 小安对白

“别急着看产品。真正的答案，要先从山里找。”

### 交互设计

* 鼠标悬停在山、雾、土壤三处，分别浮现信息卡片。
* 木牌可以点击，点击后弹出“小草坝产地名片”。
* 信息卡片不要太长，每张控制在 30—45 字。

### 动效分镜

1. 滚动进入本屏时，云雾从左右两侧缓慢掠过。
2. 山体与树木做视差移动，营造深山感。
3. 木牌从雾中显现。
4. 小安停下，抬手指向木牌。
5. 用户悬停时，信息点像萤火一样亮起。

### 素材需求

* 小草坝山林插画背景，至少分 5 层：远山、云雾、中景树林、近景灌木、前景土壤。
* 木牌元素。
* 信息点图标：山形、云雾、地理标志/定位图标。
* 小安行走动画帧或 Lottie。

## 屏 2：林下天麻生长

### 画面描述

镜头从地表缓慢下沉，进入土壤剖面。用户看到落叶层、土壤层、菌材、菌丝网络和正在生长的天麻块茎。画面可以采用“半透明剖面插画”形式。

小安蹲下，用小铲轻轻拨开落叶。

### 主文案

**它不张扬，却在地下慢慢积蓄。**

### 辅助文案

**天麻没有普通植物那样明显的叶片，它的生长离不开林下环境与菌材系统。时间，是它最重要的原料之一。**

### 信息点文案

信息点 1：**林下生长**
文案：天麻适合在阴湿、凉爽的林下环境中生长。

信息点 2：**菌材共生**
文案：天麻的生长与菌材、菌丝环境密切相关，形成独特的地下生长系统。

信息点 3：**时间沉淀**
文案：慢生长带来的不是噱头，而是原料品质形成的过程感。

### 小安对白

“看不见的地方，才是天麻真正长大的地方。”

### 交互设计

* 用户滚动时，镜头像“下潜”一样进入土层。
* 点击菌丝网络，菌丝发光，显示说明。
* 点击天麻块茎，出现“原料档案卡”。

### 原料档案卡建议内容

**原料：天麻**
**产地：昭通小草坝产区**
**形态：块茎**
**用途：食养糕点创新原料**
**备注：页面正式上线前需补充批次来源与检测信息。**

### 动效分镜

1. 地表落叶被风轻轻吹动。
2. 滚动后，镜头下潜，土壤剖面展开。
3. 菌丝以线条动画形式逐渐亮起。
4. 天麻块茎由暗到亮，成为视觉焦点。
5. 小安蹲下观察，放大镜出现。

### 素材需求

* 土壤剖面插画。
* 天麻块茎手绘图，建议准备 3 种形态：完整鲜天麻、切面、简化图标。
* 菌丝网络线条动效素材。
* 小安蹲下观察姿态。
* 原料档案卡 UI 组件。

## 屏 3：手工采挖与原料确认

### 画面描述

小安和采挖人一起从土壤中轻轻取出天麻。画面要强调“轻拿轻放”，不要做成粗放采收。右侧出现一只竹筐，里面放着饱满的鲜天麻。

用户滚动到屏幕中段时，竹筐中的天麻逐个亮起，形成“已甄选”的视觉反馈。

### 主文案

**采得慢一点，是为了把完整留给原料。**

### 辅助文案

**从山林到车间，每一枚天麻都要经过外观、完整度与基础品质的筛选。**

### 信息点文案

信息点 1：**轻采挖**
文案：减少表皮损伤，保留原料完整形态。

信息点 2：**初筛选**
文案：剔除破损、霉变或形态不合格原料。

信息点 3：**批次记录**
文案：建议正式页面接入批次编号、检测报告或供应链溯源码。

### 小安对白

“从这里开始，每一枚天麻都有自己的去向。”

### 交互设计

* 点击竹筐中的天麻，出现“甄选标准”小卡。
* 页面下方出现一条细线，作为后续旅程路线。
* 小安把一枚天麻放入竹篓，滚动后带向第二章。

### 动效分镜

1. 采挖人手部动作淡入。
2. 天麻从土中被轻轻取出。
3. 竹筐中的天麻逐个亮起。
4. 小安将天麻放入竹篓。
5. 背景山林淡出，旅程路线向下延伸。

### 素材需求

* 采挖人手部或半身插画。
* 竹筐、鲜天麻素材。
* 甄选标准 UI 卡片。
* 旅程路线 SVG。
* 小安背篓动作。

# 第二章：古法炮制 / 精细加工

## 章节定位

建立安全和工艺信任：让用户看到天麻不是被随意加入糕点，而是经过清洗、切片、烘干、粉碎等加工流程后，变成更适合糕点融合的粉体原料。

## 章节标题

**第二章 · 精细炮制**

## 章节主文案

**从一枚鲜天麻，到一撮细腻天麻粉。**

### 命名建议

原方案叫“古法炮制”，有传统感，但可能容易让用户联想到中药加工。建议正式页面使用：

**“精细炮制”** 或 **“净选精制”**

这样既保留传统感，又更适合食品页面表达。

---

## 屏 4：清洗与分拣

### 画面描述

场景切换为洁净加工间。山林中的青绿色逐渐变成米白色和浅木色。鲜天麻进入清洗水流，泥土被冲去，露出干净表皮。小安穿上围裙或工作服，站在分拣台旁。

### 主文案

**离开山林后的第一步，是洗净泥土，也洗清疑虑。**

### 辅助文案

**清洗、分拣、去除不合格原料，让进入下一道工序的每一份天麻都更可控。**

### 信息点文案

信息点 1：**清洗**
文案：去除表面泥沙与杂质，保留天麻本身形态。

信息点 2：**分拣**
文案：按照完整度、外观与使用标准进行筛选。

信息点 3：**洁净流转**
文案：建议页面展示真实车间资质、检测报告或生产批次信息。

### 小安对白

“好的食养点心，不只从好原料开始，也从干净工序开始。”

### 交互设计

* 鼠标滑过水流，泥土颗粒被“冲散”。
* 点击分拣台，显示“合格 / 剔除”的小动画。
* 可以加入“原料入库记录”样式的弹窗。

### 动效分镜

1. 第一章路线线条进入加工间。
2. 鲜天麻沿传送带进入水流。
3. 泥土颗粒随水流消失。
4. 分拣台上天麻被分成合格与不合格两组。
5. 小安点亮下一道工序。

### 素材需求

* 洁净加工间插画。
* 水流动画。
* 清洗前 / 清洗后天麻对比图。
* 分拣台、托盘、传送带。
* 小安工作服形象。

---

## 屏 5：切片、蒸制、烘干

### 画面描述

一枚完整天麻在画面中被转化为切片。切片整齐排列，进入蒸制或烘干设备。画面右侧出现时间线：“鲜天麻 → 净选 → 切片 → 烘干 → 干片”。

这一屏重点是流程感，适合横向工艺时间线，但用户仍然纵向滚动。

### 主文案

**把山里的原料，变成适合糕点的稳定风味。**

### 辅助文案

**切片、蒸制、烘干，让天麻从鲜品转化为更便于储存、研磨与应用的糕点原料。**

### 信息点文案

信息点 1：**切片**
文案：增加接触面积，让后续干燥和研磨更均匀。

信息点 2：**烘干**
文案：降低水分，提升原料稳定性，便于储存与运输。

信息点 3：**干片**
文案：干燥后的天麻片，是进入粉碎工序前的重要形态。

### 小安对白

“从完整块茎到薄片，每一步都在为更细腻的口感做准备。”

### 交互设计

* 滚动时，时间线节点逐个点亮。
* 点击任一节点，弹出该节点的简短说明。
* 鼠标移动时，切片有轻微漂浮感，增强层次。

### 动效分镜

1. 天麻从左侧进入画面。
2. 切片动画展开，像翻书一样铺开。
3. 蒸汽轻轻升起。
4. 烘干设备亮起暖光。
5. 干片落入透明容器，进入下一屏。

### 素材需求

* 天麻切片插画或照片转绘。
* 蒸汽动效。
* 烘干设备简化插画。
* 工艺时间线 UI。
* 透明容器、托盘素材。

---

## 屏 6：超微粉碎与放大镜互动

### 画面描述

画面中央是两组粉体对比：左侧“普通粉”，颗粒较粗；右侧“微米级天麻粉”，颗粒更细腻均匀。小安举起放大镜，用户移动鼠标时，放大镜跟随光标，在粉体上显示局部纹理。

这屏是第二章的核心互动点。

### 主文案

**粉得更细，是为了更好地融进一口酥香。**

### 辅助文案

**细腻粉体更适合与糕点皮馅融合，让天麻风味以更温和的方式进入传统糕点。**

### 对比卡文案

左侧：**普通粉体**
说明：颗粒感较明显，融合度相对有限。

右侧：**细腻天麻粉**
说明：粉体更均匀，更适合糕点馅料和酥皮体系。

### 注意文案

**本页展示为工艺示意，具体细度参数以实际检测数据为准。**

### 小安对白

“看，细腻不是为了炫技，而是为了让风味更自然地融进去。”

### 交互设计

* 鼠标变成放大镜。
* 放大镜经过左侧时显示粗颗粒纹理。
* 放大镜经过右侧时显示细腻纹理。
* 点击“查看对比”后，两侧粉体粒径示意图展开。

### 动效分镜

1. 干天麻片进入研磨机。
2. 粉体从设备下方缓缓落下。
3. 左右两组粉体分别铺开。
4. 小安举起放大镜。
5. 用户移动鼠标，放大镜局部放大颗粒。
6. 对比完成后，粉体汇入一束金色光线，流向第三章车间。

### 素材需求

* 普通粉 / 细腻粉体纹理图。
* 放大镜 SVG 或 Canvas 互动组件。
* 粉碎设备插画。
* 粉体颗粒动效。
* 对比弹窗 UI。
* 小安举放大镜姿态。

# 第三章：百年手艺

## 章节定位

让用户明白：这款产品不是把天麻“硬塞”进糕点，而是由月中桂的老字号糕点技艺完成现代转译。

## 章节标题

**第三章 · 百年手艺**

## 章节主文案

**一味天麻，走进一门百年糕点手艺。**

---

## 屏 7：进入月中桂车间

### 画面描述

从第二章的洁净加工间转场到月中桂烘焙车间。画面色调由米白转为暖黄色。门头或墙面上出现“月中桂”字样，旁边有老照片、木制模具、糕点印章等元素。

小安推开车间门，暖光从门缝里透出。

### 主文案

**山里的天麻，来到月中桂。**

### 辅助文案

**近百年的糕点经验，让食养原料有了更温和、更日常的表达方式。**

### 品牌背书卡文案

**月中桂**
昭通老字号糕点品牌。以传统糕点制作技艺与地方风味产品为重要品牌资产。

### 小安对白

“接下来，就要把原料交给真正懂糕点的人了。”

### 交互设计

* 点击墙上的老照片，出现品牌时间轴。
* 点击木制糕点模具，出现“传统糕点技艺”说明。
* 点击月中桂招牌，出现品牌名片。

### 动效分镜

1. 第二章的金色粉体线条飞入门缝。
2. 车间门缓缓打开。
3. 暖光扩散，背景音乐从山林音转为烘焙车间轻音乐。
4. 墙上的老照片依次亮起。
5. 小安进入画面，引导用户继续向下。

### 素材需求

* 月中桂品牌门头 / 招牌授权素材。
* 老照片风格插画或真实照片。
* 车间暖光背景。
* 糕点模具、擀面杖、木桌、烤盘素材。
* 品牌时间轴 UI。

## 屏 8：揉面、制馅、包酥

### 画面描述

这是第三章核心屏。老师傅在木案前操作，画面分为三段动作：揉面、调馅、包酥。天麻粉以细腻金色粉末的形式融入馅料或面团中。

画面要突出手部细节，避免过度工业化。

### 主文案

**把食养智慧，揉进一层层酥香里。**

### 辅助文案

**天麻的清雅，遇见酥皮的麦香、馅料的细腻，成为更适合当代人的东方食养糕点。**

### 工艺点文案

工艺点 1：**揉面**
文案：面团反复揉合，形成稳定而柔韧的基础。

工艺点 2：**调馅**
文案：天麻粉与馅料均匀融合，让风味不突兀。

工艺点 3：**包酥**
文案：层层包裹，烘焙后形成外酥内润的口感。

### 可替代文案

如果“32道工序”有官方依据：
**一块天麻酥，32道工序，百年手艺不变。**

如果没有官方依据，建议用：
**一块天麻酥，层层工序，承接百年糕点手艺。**

### 小安对白

“好的创新，不是把传统推翻，而是让传统多走一步。”

### 交互设计

* 用户滚动时，三个工艺动作依次展开。
* 鼠标悬停在师傅手部，出现工艺点说明。
* 点击天麻粉，粉末以金色粒子形式融入馅料。
* 可以加入“拖拽天麻粉入馅料”的轻互动，但开发成本会稍高。

### 动效分镜

1. 老师傅双手揉面，面团轻微变形。
2. 天麻粉从小安手中的容器缓缓撒下。
3. 馅料旋转搅拌，金色粉末融入。
4. 面皮展开，馅料包入。
5. 酥饼成型，印章轻轻盖下。
6. 镜头跟随烤盘进入下一屏。

### 素材需求

* 老师傅形象插画，重点是手部动作。
* 揉面、调馅、包酥三段动作素材。
* 天麻粉粒子动效。
* 面团、馅料、酥皮、成型饼胚素材。
* 月中桂糕点模具或印章元素。
* 工艺说明浮层 UI。

### 画面描述

### 辅助文案

**一口酥香，既有昭通天麻的原料记忆，也有月中桂糕点手艺的温度。**

文案：层层酥皮带来轻盈口感。

信息点 2：**内馅细腻**
文案：细腻天麻粉与馅料融合，风味温和。

文案：把东方食养智慧，转译成日常可分享的点心。

### 小安对白

* 用户滚动到烤炉时，炉门自动打开。

2. 屏幕短暂变暗，只保留炉内暖光。
3. 烤炉门打开，暖光铺满屏幕。
4. 香气线条向上扩散。
5. 天麻酥剖面弹出，展示酥皮与内馅。
6. 画面过渡到礼盒场景。

### 素材需求

* 烤炉插画 / 轻 3D 模型。
* 烤盘、天麻酥成品图。
* 天麻酥剖面图。
* 香气线条动效。

# 五、全页面动效设计总表

## 1. 滚动主线动效

### 形式

使用长页面滚动触发动画。小安作为固定引导角色，随着章节切换改变姿态和服装。

### 技术建议

* 使用 GSAP + ScrollTrigger 实现滚动触发。
* 章节之间用 pin 固定画面，用户滚动时播放内部动画。
* 山林、土壤、车间、礼盒均采用分层 SVG / PNG 实现视差。

### 重点效果

1. **首屏云雾视差**
2. **土壤剖面下潜**
3. **天麻采挖发光确认**
4. **粉体放大镜对比**
5. **车间门打开转场**
6. **老师傅手作分步动画**
7. **烤炉出炉暖光扩散**
8. **礼盒展开与信任标签环绕**

---

## 2. 交互点设计

### 每章建议 2—3 个交互点

不要每屏都堆满互动，否则用户会疲惫。重点互动应集中在：

1. 第一章：产地信息点悬停。
2. 第二章：放大镜查看粉体细度。
3. 第三章：点击工艺动作查看说明。
4. 第四章：点击信任标签回顾旅程。

### 交互反馈原则

* 悬停：小卡片浮现。
* 点击：弹出更完整信息。
* 滚动：自动推进故事。
* 完成互动：标签点亮，形成成就感。

---

## 3. 转场设计

### 第一章 → 第二章

天麻从土中被采出，进入竹篓。竹篓中的天麻变成一条金色路线，带用户进入清洗工序。

### 第二章 → 第三章

天麻粉从研磨设备中落下，化为金色粒子，穿过画面进入月中桂车间。

### 第三章 → 第四章

刚出炉的天麻酥从烤盘进入礼盒，完成“产品化”。

# 十一、开发实现建议

## 技术栈

* HTML / CSS / JavaScript。
* GSAP + ScrollTrigger：主滚动动画。
* Lottie：小安角色动作、轻量图标动画。
* SVG：路线、菌丝、香气线条、信任标签。
* Canvas：放大镜粉体对比效果。

## 页面性能建议

* 所有大图压缩为 WebP。
* 首屏优先加载，后续章节懒加载。
* 动画素材尽量用 SVG / Lottie，减少视频。
* 移动端简化视差和复杂动画。

## 移动端适配

移动端不建议做复杂鼠标悬停，改成点击触发。

移动端交互替代：

* 悬停信息点 → 点击信息点。
* 鼠标放大镜 → 手指拖动放大镜。
* 横向时间线 → 纵向卡片。
* 礼盒旋转 → 左右滑动切换角度。

# 十二、最终建议

这页的关键不是把所有事实都塞进去，而是让用户形成一句清晰感受：

**我知道这块天麻酥从哪里来，也知道它为什么值得相信。**

所以页面的视觉重点应该永远围绕三件事：

1. **来处可信：昭通小草坝。**
2. **过程可信：清洗、分拣、切片、粉碎。**
3. **成品可信：月中桂老字号糕点手艺。**

只要这三点成立，这个一页式网页就不仅是视觉展示，而是完整的品牌信任体验。
